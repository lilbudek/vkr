\abstractRU
{
В первой главе были рассмотрены семейства микроконтроллеров, различные среды разработки и изучены методы генерации сигналов.

Во второй главе был спроектирован генератор. Создана схема электрическая принципиальная, проведено моделирование методов генерации, разработан алгоритм работы.

В третьей главе была произведена сборка макета, написана и протестирована результирующая программа. 
}
\abstractEN{
In the first part, families of microcontrollers, various development environments were considered and signal generation methods were studied.

In the second part, a generator was designed. An electrical schematic diagram has been created, generation methods have been modeled, and an algorithm has been developed.

In the third part, the layout was assembled, the resulting program was written and tested.
}
